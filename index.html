Classes

This semester I am taking the following classes:

CMPSC 3323 – Internet Programming

This course covers web development using HTML, CSS, and JavaScript. I really enjoy it because it teaches how to build real websites from scratch, and the hands-on projects make it easy to see how everything connects together.

CMPSC 3313 – Adv Computer Lang-Java

This is an advanced Java programming course that dives into object-oriented design, data structures, and algorithms. I like how it challenges me to think logically and write cleaner, more efficient code.

CMPSC 4213 – Data Base Design

This course explores how to design, create, and query relational databases using SQL. It's really practical because almost every application needs a database, so this knowledge is super useful.

CMPSC 3253 – Introduction to Cloud Security

This class introduces cybersecurity concepts in cloud computing environments. I find it fascinating because cloud security is a growing field and understanding how to protect data in the cloud is incredibly relevant today.

CMPSC 4473 – Theory Of Programming Languages

This course digs into the theory behind how programming languages are designed and implemented. It's challenging but rewarding because it helps me understand what's actually happening under the hood of any language I use.
